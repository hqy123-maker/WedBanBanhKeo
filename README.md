🍬 E-commerce Candy Store (Fullstack Web Application)

📌 Introduction

This project is a fullstack e-commerce web application for selling candy and confectionery products.
It includes both frontend and backend systems, supporting user authentication, product management, and order processing.

---

🚀 Features

🔐 Authentication

- User registration & login
- JWT-based authentication
- Password hashing using bcrypt

🛍️ Product Management

- View product list
- Product details
- Admin: Add, update, delete products

🛒 Shopping Cart & Orders

- Add/remove products to cart
- Checkout functionality
- Order management

🔎 Other Features

- Product search & filtering
- Responsive UI (mobile & desktop)

---

🛠️ Tech Stack

Backend

- Node.js (Express)
- RESTful API
- JWT Authentication
- MVC Architecture

Database

- MySQL

Frontend

- Vue.js
- JavaScript

Tools

- Postman (API testing)
- Git & GitHub

---

⚙️ Installation & Setup

1. Clone repository

git clone https://github.com/your-username/your-repo.git
cd your-repo

2. Install dependencies

npm install

3. Configure environment variables

Create a ".env" file:

PORT=3000
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=yourpassword
DB_NAME=ecommerce
JWT_SECRET=your_secret_key

4. Run server

npm start

---

🧪 Testing

Use Postman to test API endpoints.
